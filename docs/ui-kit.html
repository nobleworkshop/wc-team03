Документация по фреймворку

Это PUG LESS фреймворк для быстрой верстки проектов. Он содержит в себе все необходимые элементы страницы. Каждый блок или секция имеют готовую pug разметку и less стили. На отдельной странице каждого блока можно найти код его верстки для использования.
